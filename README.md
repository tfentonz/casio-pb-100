# The Casio PB-100—Learn as You Go

This repository contains program listings from a programming book for
the Casio PB-100 pocket computer.

The programs are organized by chapter to match the structure of the
original book.

## Chapters

- [Chapter 1 What Is the PB-100 Menu Used for?](chapter-1/README.md)
- [Chapter 2 What Is a Program ?](chapter-2/README.md)
- [Chapter 3 This Is a Program !](chapter-3/README.md)
- [Chapter 4 Now Let's Learn BASIC.](chapter-4/README.md)
- [Chapter 5 Program Library](chapter-5/README.md)

## Notes

- Programs are written in PB-100 series BASIC.
- Line numbers are preserved from the original listings.
